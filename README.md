# okky 클론 사이트 프로젝트 🖥
<hr/>

## 프로젝트 개요
### 1. 프로젝트 주제 및 선정 배경, 기획 의도
- 개발자들간 의사소통이 용이한 사이트 구현
- 개발자들의 정보 공유가 활발한 사이트 구현
- 개발자들의 구인구직이 용이한 사이트 구현
  <br> ➡ OKKY사이트 벤치 마킹

### 2. 프로젝트 내용
- 게시판 구조의 MVC모델을 활용한 커뮤니티 사이트 구현
